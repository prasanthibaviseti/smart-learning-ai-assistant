# 🤖 AI Doubt Solving + Personalized Learning System

## 📌 Overview
This project is an AI-powered doubt solving system built using Retrieval-Augmented Generation (RAG). It retrieves relevant knowledge chunks using embeddings and generates accurate, context-aware answers.

## 🚀 Features
- 🔍 Semantic search using embeddings
- 📚 Knowledge retrieval (RAG pipeline)
- 🤖 AI-generated answers
- 🎯 Topic detection & analysis
- 📊 Adjustable context chunks

## 🛠 Tech Stack
- Python
- Sentence Transformers (all-MiniLM-L6-v2)
- Streamlit / Flask (based on your app)
- NumPy, Pandas

## ⚙️ How It Works
1. User enters a question
2. Question is converted into embeddings
3. Relevant chunks are retrieved from dataset
4. AI generates answer using retrieved context
